# Titanic Data Visualization & Analysis

This project performs **exploratory data analysis (EDA)** on the Titanic dataset.  
The goal is to understand survival patterns using visualizations.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

##  Key Visualizations
- Age distribution
- Survival counts
- Survival by gender
- Survival by passenger class
- Heatmap (correlation)

##  Purpose
A professional-level data analysis notebook demonstrating  
**data cleaning, visualization, insight generation**, and  
real-world data exploration.
